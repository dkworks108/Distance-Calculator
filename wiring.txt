Simple ASCII wiring blueprint (top-down view)

  [HC-SR04]
  +---------------+
  |  VCC  TRIG     |
  |  GND  ECHO     |
  +---------------+

Wiring to Arduino Uno pins:

HC-SR04 VCC  ---> Arduino 5V
HC-SR04 GND  ---> Arduino GND
HC-SR04 TRIG ---> Arduino D9
HC-SR04 ECHO ---> Arduino D10

Notes:
- Use short jumper wires for better stability.
- If using a 3.3V microcontroller, place a voltage divider on ECHO (HC-SR04 is 5V logic).
- Place sensor perpendicular to the target for best accuracy.
